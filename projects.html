IMU & Microcontroller Data Logger

4-layer STM32F4-based embedded system with MPU-6050 IMU for real-time motion data acquisition via I2C. Controlled-impedance PCB design (90Ω USB), LDO power regulation (5V to 3.3V), and end-to-end firmware integration.

Smart Home RF Access Control System

Modernized legacy RF access control hardware with an ESP32-based IoT solution featuring a custom Android interface. Deployed across 30+ residential installations with zero reported failures.

PID-Controlled AC Heating System

Designed a PID-controlled AC heating system using phase-angle control with zero-crossing detection. 4-layer Altium PCB with PT500 sensor accuracy, LabVIEW real-time monitoring, and 50ms data logging. Built at B/S/H/.

Circular Economy in PCB Design

Developed PCB design guidelines spanning the full lifecycle (design, use, recycling) as part of the DEEP-Opt joint project at FAPS, in collaboration with B/S/H/ and multiple industrial partners. Awarded grade 1.7 (gut).
